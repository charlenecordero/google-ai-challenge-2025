## ‚ö° Deployment & Automation
The project utilizes a fully automated **Uplink** from GitHub to Google Cloud.

### Automated CI/CD Pipeline
Every push to the `main` branch triggers a multi-stage **Cloud Build** process:
1.  **Build:** Compiles the container using `cloudbuild.yaml`.
2.  **Push:** Securely tags and uploads images to the **Artifact Registry** (`asia-southeast1`).
3.  **Deploy:** Automatically updates the **Cloud Run** service with zero downtime.
