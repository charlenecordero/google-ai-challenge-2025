>> Codebase Overview

SYSTEM_LOG: This portfolio is built as a lightweight Single Page Application (SPA) leveraging modern frontend technologies without heavy framework overhead.

TECHNOLOGY STACK

Node.js & Express

Robust backend runtime serving the application and acting as the secure gateway to the Gemini API layer.

Google Gemini

Generative AI model (gemini-1.5-flash) powering the interactive "Aura" chat interface and dynamic content generation.

Google Cloud Run

Serverless container platform ensuring auto-scaling, high availability, and zero-maintenance infrastructure.

Alpine.js

Lightweight reactive framework managing frontend state, transitions, and user interactions without virtual DOM overhead.

Tailwind CSS (CLI)

Utility-first design system utilizing Tailwind CLI for custom build steps and verified local class generation.

Chart.js

Renders responsive, hardware-accelerated data visualizations for career and skill metrics.

// 1. Frontend: Sends Message to Backend
async function sendMessage(msg) {
    const response = await fetch('/api/chat', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message: msg })
    });
    return await response.json();
}

// 2. Backend (server.js): Calls Gemini
