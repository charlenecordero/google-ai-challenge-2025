SYSTEM_LOG: This portfolio is built as a lightweight Single Page Application (SPA) leveraging modern frontend technologies without heavy framework overhead.

TECHNOLOGY STACK

Alpine.js

Handles reactive state management, routing (via View swapping), and UI interactions like the dynamic calendar and modal logic.

Tailwind CSS

Utility-first styling framework enabling the custom "Cyberpunk x Cottagecore" design system through arbitrary values.

Semantic HTML5

Accessible, structured markup ensuring SEO optimization and proper document outline for screen readers.

Chart.js

Powers the "Career Timeline" visualization in the dashboard, rendering responsive canvas-based graphs.
